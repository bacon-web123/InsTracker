全新功能：IG 專頁貼文照片「一鍵打包下載 (ZIP)」

想要下載某位朋友、網紅或自己專頁裡的所有貼文相片嗎？我們為您撰寫了專用下載腳本，會自動抓取該用戶專頁上的**所有貼文（含多圖輪播）並打包成 ZIP 下載**，相片將依照日期與序號命名。

1

在瀏覽器開啟目標用戶的 IG 專頁 （例：https://www.instagram.com/目標帳號/）

請確保您是在登入狀態下，並停在該帳號的個人首頁。

2

複製下方「貼文相片打包下載腳本」

點擊右側複製按鈕，這段安全、高效的媒體下載核心代碼會存入您的剪貼簿。

3

貼上執行，開始自動打包！

在該 IG 頁面上按下鍵盤的 F12 (Mac 請按 Option + Cmd + I)。 切換到 Console (控制台) 分頁，並按下 Ctrl + V 貼上後按 Enter。

4

大功告成，收取 ZIP 檔

腳本會顯示進度條，將所有相片快速抓取，並自動下載一個名為 IG_貼文照片_[帳號].zip 的檔案。解壓縮後即可獲得該專頁的所有高畫質相片！

安全與隱私權宣告

  • 免輸入密碼：此操作完全在您登入的瀏覽器本機內進行，不會外洩任何個人權限資訊。
  • 支援任何可看帳號：只要您的 IG 帳號本身可以看得到的個人專頁（包含公開帳號、及您已追蹤的私人帳號），通通都能順暢抓取！
  • 抗封鎖延遲：內建防頻繁抓取延遲，安全平穩。
溫馨提示：本功能亦支援下載限時動態精選，或是貼文內的所有短片封面照片。

超簡單！獨家「100% 真實粉絲資料」極速抓取步驟

不需要提供密碼給任何可疑網站！只需打開您平常用的網頁瀏覽器，複製下方特製的抓取代碼，在 Instagram 網頁版上貼上執行即可。100% 安全、不封號。

1

登入 Instagram 網頁版 開啟 IG 網站

請在電腦的 Chrome / Edge 瀏覽器登入您的 IG 帳號。

2

複製下方「一鍵真實關係抓取代碼」

點擊右側複製按鈕，我們為您準備的無毒、安全 API 呼叫指令就會存入剪貼簿。

3

貼上並執行

在 IG 網頁的任何地方按下鍵盤的 F12 (Mac 請按 Option + Cmd + I) 開啟開發者控制台，切換到 Console (控制台) 分頁，並按下 Ctrl + V 貼上後按 Enter 執行。

4

輸入帳號與貼回結果

腳本會彈出視窗詢問要抓取的帳號，並在背景快速抓取真實名單，抓完後會自動幫您複製。您只需回到這裡（右側），在輸入框內按貼上，大功告成！

備用方案：使用官方下載的 ZIP 資料包匯入

1. 到 IG App 申請下載「個人資料」(格式選 JSON、資料類型選：追蹤關係)。

2. 解壓縮後在 connections/followers_and_following/ 可以找到：

• followers_1.json (粉絲名單) 與 following.json (追蹤中名單)
